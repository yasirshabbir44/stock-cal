Privacy & Open Source

StockCal stores all portfolio data locally in your browser. There are no accounts, no analytics, and no tracking scripts. The full source code is public — you can inspect every line yourself.

  • Portfolio data stays on your device (IndexedDB)
  • No third-party analytics or ad trackers
  • Optional Finnhub API calls only when you add a key
